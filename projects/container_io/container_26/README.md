# 容器

请利用 java 容器完成以下问题。

对于每个问题，请先思考利用哪个容器可以更好的抽象问题。

## 实验内容

1. 你现在在管理一个具有实时收益的直播间，直播总时长为`n`分钟，第`i`分钟的收益可以表示为 $a_i$ 。作为直播间的管理员，你需要对一些时间段的总收益进行分析从而调整直播内容，所以你想快速的获得任意时间段 `[l, r]` 之间的直播收益总和。

   上述问题可以形式化为：

   给定长度为 `n` 的整数数组 `a`，以及 `q` 次查询。每次查询给出区间 `[l, r]`，请输出该区间元素之和（保证时间复杂度为 **O(1)**，不使用循环语句 ）：
   $$
   a_l + a_{l+1} + \dots + a_r
   $$
   **输入格式**

   第一行：两个整数 `n q`
   第二行：`n` 个整数 `a1 a2 ... an`
   接下来 `q` 行：每行两个整数 `l r`

   **输出格式**

   对每个查询输出一行区间和。

   **输入样例**

   ```
   5 4
   1 2 3 4 5
   1 3
   2 5
   4 4
   1 5
   ```

   **输出样例**

   ```
   6
   14
   4
   15
   ```

   ------

   **提示：**考虑使用数组的前缀和，设：

   - `pre[0] = 0`
   - `pre[i] = a1 + ... + ai`（i 从 1 到 n）

   则查询转化为：

   - `sum(l, r) = pre[r] - pre[l-1]` 

   **思考：**在此基础上，如果每分钟直播的成本为`k`，哪如何选择直播时间段才能使得你最终的总收益最多？

2. 现在有一个电子翻译器，可以自动翻译内存中的单词，并且可以查询字典获取内存的没有的单词翻译

   电子翻译器的具体描述如下：

   其内存容量 为`M` ，要翻译的文章长度为 `N`，文章由 `N` 个单词组成，编号由1到N。

   - 初始内存为空，最多存 `M` 个**不同**单词。
   - 依次处理文章的每个单词 `x`：
     - 若 `x` 已在内存中：不查词典。
     - 若 `x` 不在内存中：需要 **查词典次数 +1**，并把 `x` 放入内存。
     - 若内存已满，则先删除 **最早进入内存** 的那个单词，再放入 `x`。

   你需要输出整个过程中需要查词典的总次数。

   **输入格式**

   共 $2$ 行。每行中两个数之间用一个空格隔开。

   第一行为两个正整数 $M,N$，代表内存容量和文章的长度。 

   第二行为 $N$ 个非负整数，按照文章的顺序，每个数（大小不超过 $1000$）代表一个英文单词。

   文章中两个单词是同一个单词，当且仅当它们对应的非负整数相同。

   **输出格式**

   一个整数，为软件需要查词典的次数。

   **输入样例**

   ```
   3 7
   1 2 1 5 4 4 1
   ```

   **输出样例**

   ```
   5
   ```

3. 公司派你组装一套系统，该系统需要安装两个特殊模块。因为经营不周，公司财务没有钱去买全新的模块，只能从仓库中找出N个不同型号（标号为1-N）的模块供你挑选，每个型号的模块性可以用一个整数表示。由于系统适配要求，你挑选出第一个模块之后，只能再挑选标号在其之后的模块，且第一个模块和第二个模块的性能之差必须恰好为一个指定的整数。你需要在这一堆模块中尝试挑选出可能的组合，并告诉老板所有可能的组合数量是多少。

   该问题可以形式化为：

   给出一串正整数数列以及一个正整数`C`，要求计算出所有满足`A−B=C`的数对的个数（不同位置的数字一样的数对算不同的数对）。

   因为公司库存中的模块实在太多，需要快速决定，因此你的老板要求你的代码中不许使用两层循环。

   **输入格式**

   输入共两行。

   第一行，两个正整数 $N,C$。

   第二行，$N$ 个正整数，表示每一个模块的性能。

   **输出格式**

   一行，表示该串正整数中包含的满足 $A - B = C$ 的数对的个数。

   **输入样例**

   ```
   4 1
   1 1 2 3
   ```

   **输出样例**

   ```
   3
   ```

4. 挑选好准备使用的模块之后，你打算开始构建系统。该系统有n个部件，和很多很多篇文档，每一个部件`i`的使用说明在 $doc_i$ 中，可能多个部件的说明都在同一篇文档中。现在你要看所有部件的使用说明，所以需要统计一下有那些文档是需要看的。

   该问题可形式化为：

   给定 *n* 个数，要求把其中重复的去掉，只保留第一次出现的数。

   **输入格式**

   多组数据，第一行一个整数 $T$，表示数据组数。

   对于每组数据：

   第一行一个整数 $n$。

   第二行 $n$ 个数，表示给定的数。

   **输出格式**

   对于每组数据，输出一行，为去重后剩下的数，两个数之间用一个空格隔开。

   **输入样例**

   ```
   2
   11
   1 2 18 3 3 19 2 3 6 5 4
   6
   1 2 3 4 5 6
   ```

   **输出样例**

   ```
   1 2 18 3 19 6 5 4
   1 2 3 4 5 6
   ```
